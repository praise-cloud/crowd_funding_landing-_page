## Final Architecture

### React Router Structure
```
<Router>
  <ScrollToTop />
  <Routes>
    <Route path="/" element={<Layout />}>
      <Route index element={<HomePage />} />
    </Route>
  </Routes>
  {/* Modals */}
</Router>
```

### Navigation System
- Hash-based routing: `#introduction`, `#funding`, `#programs`, `#impact`, `#donate`
- Smooth scrolling with the useScrollTo hook
- URL management without reloads
- Scroll restoration with ScrollToTop

### Motion System
- `useScrollReveal` hook for intersection-based reveals
- `.reveal` and `.is-visible` classes for smooth section entry
- `.micro-float` and `.cinematic-zoom` for ambient motion
